The development version of PsN is available from the GitHub repository. Remember that stability is not guaranteed for the development version - it might be broken and it might break at any time in the future. See the Developer's guide.pdf for more information on how to install from the git repository.

Compatibility

Grid/Cluster support: PsN has built-in functionality for submitting the NONMEM runs to SGE, Slurm, LSF, ud and torque. See documentation. The PsN test suite has been run on slurm and locally.
